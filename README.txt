Language Used: Python 3.7.4

When using ChessMoves.py, enter the pieces as "Rf1, Kg1, Pf2" where the peices are seperated by a ", ". The
Piece to move must be one of the pieces listed in the black or white pieces. Example inputs can be found in
TestCases.txt.

K = King
Q = Queen
R = Rook
B = Bishop
N = Knight
P = Pawn

The program does not check if a move would leave a friendly King in check.
